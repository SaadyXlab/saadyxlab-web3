# 🧪 Onchain UX Lab

A collection of experimental Web3 tools focused on user behavior, interaction, and onchain identity.

## 🚀 Projects

### 🔹 Chain Rep Score
A wallet-based identity tool that:
- Connects wallet
- Analyzes onchain signals (ETH balance)
- Assigns a reputation score
- Allows sharing results

### 🔹 Mint War Room
A simulated mint dashboard that explores:
- Live mint activity
- Gas fluctuations
- FOMO mechanics
- User interaction patterns

---

## 🧠 Goal

To understand how users interact with Web3 products beyond just design — focusing on:
- Psychology
- Real-time feedback
- Trust signals

---

## ⚙️ Tech Stack

- HTML / CSS / JavaScript
